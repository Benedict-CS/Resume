## 🛠️ How to Build
This resume is based on a modern LaTeX class (`russell.cls`) and requires the `XeLaTeX` compiler due to the usage of specific fonts (like Roboto and FontAwesome).

To compile the resume locally, run the following command:

```bash
xelatex resume.tex
